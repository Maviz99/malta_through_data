# 🇲🇹 Malta Through Data – A Personal Travel Dashboard

## 🌟 Project Overview
This is a personal passion project born out of my enthusiasm for mathematics, data, and travel. Next month, I am traveling to Malta with my sister. To prepare for the trip and practice my data engineering skills, I am building an interactive dashboard to plan our journey, analyze locations, and tell our travel story through data.

This project is a **living repository**. It starts today (June 2026) as a pre-trip planner and will evolve during and after our journey into a full digital travel log.

## 🎯 Learning Goals
* **Data Storytelling:** Translating personal travel experiences and geographical data into a cohesive visual story.
* **Frameworks & Libraries:** Gaining hands-on experience with `Streamlit`, `Pandas`, `NumPy`, and data visualization (`Matplotlib`/`Seaborn`).
* **Software Architecture:** Moving from theoretical university concepts to building a practical, user-centric application.

## 🗺️ Roadmap & Features

### 🔹 Phase 1: Pre-Trip Planning (Current Status)
* [x] Set up a virtual environment and project structure.
* [x] Create a base dataset of Maltese cities using **Pandas DataFrames**.
* [x] Integrate interactive **Streamlit** maps to visualize coordinates.
* [x] Implement **NumPy** logic to calculate walking distances from our hotel.

### 🔸 Phase 2: Itinerary & Travel Integration (Coming Soon)

* [ ] Integrate budget tracking (food, transport, entry tickets).
* [ ] Implement a restaurant and beach scoring system.

### 🔸 Phase 3: Post-Trip Travel Log & Memories (After Return)
* [ ] Upload personal photos and embed them directly into the dashboard.
* [ ] Create a digital travel diary mapped to specific locations.
* [ ] Analyze our actual trip expenses vs. our planned budget.
